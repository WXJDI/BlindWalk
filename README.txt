TITRE : BLIND WALK - Simulation de Cécité Sélective
AUTEUR : Wajdi Ben Ouirane (BUT3 INFO)

--- PRÉREQUIS ---
- Python 3.10 ou supérieur
- Bibliothèque Pygame

--- INSTALLATION ---
1. Ouvrez un terminal dans le dossier du projet.
2. Installez les dépendances avec la commande :
   pip install -r requirements.txt

--- LANCEMENT ---
1. Exécutez le fichier principal :
   python main.py

--- COMMANDES EN JEU ---
- Flèches directionnelles : Se déplacer
- 1 : Mode Deutéranopie (Cache le rouge)
- 2 : Mode Tritanopie (Cache le bleu)
- 3 : Mode Achromatopsie (Cache l'orange)
